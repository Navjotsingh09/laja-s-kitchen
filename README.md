# Laja Kitchen 🍛

A vibrant, modern website for Laja Kitchen - showcasing authentic Indian street food with a contemporary design inspired by the vibrant streets of Delhi.

## 🌟 Features

- **Responsive Design**: Fully optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Clean, vibrant interface with smooth animations
- **Interactive Elements**: Smooth scrolling, hover effects, and dynamic content
- **Brand Identity**: Consistent use of brand colors and Hindi typography
- **Performance Optimized**: Fast loading and smooth user experience

## 🎨 Design Inspiration

The website design is inspired by My Delhi Street Food, featuring:
- Vibrant color palette reflecting Indian street food culture
- Hindi text headings (Devanagari script)
- Modern card-based menu layout
- Elegant typography combining Playfair Display and Poppins fonts
- Engaging hero section with call-to-action buttons

## 🚀 Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display & Poppins)

## 🎯 Sections

1. **Hero Section**: Eye-catching landing with brand message
2. **About**: Introduction to Laja Kitchen's story
3. **Menu**: Comprehensive food menu with categories:
   - Street Food Favourites
   - Main Courses
   - Tandoor Specialties
4. **Awards**: Showcase of achievements
5. **Contact**: Contact information and form
6. **Footer**: Quick links and social media

- Desktop: 1400px+
- Tablet: 968px - 1399px

## ⚡ Performance Features

- Optimized CSS with minimal dependencies
- Intersection Observer for scroll animations
- Smooth scroll behavior
- Lazy loading ready
- Mobile-first approach

## 🔧 Setup & Installation

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!
